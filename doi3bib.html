doi3bib

Updated on August 16, 2023
This project aims to create a robust platform that converts object identifiers into bibtex entries compatible with LaTeX. It is a simple javascript implementation that takes a DOI and converts it to a bibtex entry received from either doi.org or crossref.org. It parses that entry for known errors and corrects for them so they work correctly with LaTeX.

doi3bib is accessible at https://doi3bib.com.
